# Teste HOX

This aplication is a teste for HOX.

## Instalation

Use [yarn](https://yarnpkg.com/) or [npm](https://www.npmjs.com/) to install all dependencies to this project.

```bash
yarn
```

```bash
npm install
```

After that use 'yarn dev' or 'npm run dev' to run the application.

```bash
yarn dev
```

```bash
npm run dev dev
```

## Technologies

- TypeScript
- React.JS
- Next.JS
- Redux
- TailwindCSS

## Features

- Login System.
- CRUD for Products.
